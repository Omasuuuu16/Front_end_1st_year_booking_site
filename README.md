# Front_end_1st_year_booking_site

## About the Project

This was one of my first front-end projects during my first year at university. It is a simple booking website built as a team project.

The project is **not perfect**, and I know there are many things that could be improved. At that time, we were still learning the basics of web development, and our teamwork was not as organized as it could have been.

## My Contribution

I was responsible for the **Middle East** section of the website. It was the part I spent the most time on, and I believe it became the best-looking section in the project.

Although the overall project had room for improvement, my contribution helped the team earn a **full grade (100%)** from our instructor.

## Technologies Used

* HTML
* CSS
* JavaScript
